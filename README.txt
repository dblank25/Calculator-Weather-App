How to Open Site:
  1. Execute "node Server.js" from terminal to start server
  2. In browser, go to url http://localhost:8080/

How to Use this Site:
  1. Click menu panel on left to access Calculator & Weather Apps
  2. Calculator App: Choose summation or factorial from the drop down menu and input number to calculate
  3. Weather App: Click button on page to display weather after allowing browser to use your current location
  4. Click on home button in menu panel to go to home page
